# Governance Filtering Specification

## Overview

The middleware filters:
unauthorized semantic memory
before:
retrieval ranking.

Governance filtering applies to:

* retrieval
* traversal
* compression
* summaries
* analytics

Governance enforcement is:
pre-execution infrastructure behavior.

---

# Core Philosophy

Unauthorized memories should never:
participate in ranking pipelines.

The middleware prioritizes:
secure semantic execution.

---

```txt id="gf1"
Request

```

# Filtering Philosophy
