# Query Engine Philosophy

## Purpose

Transform raw user input into:

- intent
- entities
- semantic signals
- retrieval strategy
- compression requirements

before retrieval begins.

---

# Core Principle

The retrieval engine should never receive:

raw user text.

It should receive:

structured retrieval instructions.

---

# Query Pipeline

User Query
↓
Intent Classification
↓
Entity Extraction
↓
Query Decomposition
↓
Retrieval Planning
↓
Retrieval Execution

---

# Philosophy

Understanding the question is often more valuable than retrieving more context.
