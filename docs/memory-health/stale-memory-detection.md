# Stale Memory Detection Specification

## Overview

The middleware detects:
stale semantic memory.

Stale memories may include:

* outdated workflows
* obsolete pricing
* expired policies
* low-value historical context

Stale memory affects:
retrieval precision.

---

# Core Philosophy

Not all memories should retain:
equal retrieval influence forever.

The middleware prioritizes:
fresh high-value semantic memory.

---

# Staleness Signals

Examples:

* age
* low retrieval frequency
* failed retrieval outcomes
* version supersession
* weak reinforcement

Staleness detection remains:
deterministic and observable.

---

# Stale Memory Handling

Recovery systems may trigger:

* decay
* archival
* revalidation
* summary regeneration

Handling remains:
governed and reversible.

---

# Staleness Philosophy

The middleware values:
active semantic relevance,
not infinite memory accumulation.
