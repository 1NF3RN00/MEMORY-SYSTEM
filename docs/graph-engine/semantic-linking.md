# Semantic Linking Specification

## Overview

The middleware generates:
semantic graph links between memories.

Semantic linking connects:
conceptually related memories.

Semantic links improve:

* retrieval enrichment
* graph continuity
* semantic clustering
* contextual reconstruction

---

# Core Philosophy

Semantic memory exists in:
conceptual relationships.

The middleware treats:
semantic links
as retrieval infrastructure.

---

# Semantic Link Signals

Semantic linking may consider:

* embedding similarity
* phrase overlap
* metadata overlap
* workflow continuity
* source similarity

Semantic linking remains:
deterministic-first.
