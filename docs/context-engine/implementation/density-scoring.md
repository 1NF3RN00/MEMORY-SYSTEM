# Semantic Density Scoring Specification

## Overview

The middleware evaluates:
semantic density.

Semantic density measures:
meaning retained per token.

The middleware prioritizes:
high-density semantic contexts.

---

# Core Philosophy

Large context windows are often:
low-density semantic noise.

The middleware values:
compact,
high-information semantic working memory.

Semantic density is:
a core optimization target.

---

# Density Formula

```txt id="ds1"
SemanticDensity =
(
  MeaningRetained
)
/
(
  TokenCount
)
```

---

# Density Signals

Density scoring may consider:

* redundancy
* overlap
* semantic uniqueness
* retrieval confidence
* source authority

High-density contexts are prioritized.

---

# Compression Relationship

Compression attempts to:
increase semantic density.

The middleware values:
maximum semantic intelligence per token.
