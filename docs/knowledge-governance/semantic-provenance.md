# Semantic Provenance Specification

## Overview

The middleware preserves:
semantic provenance metadata.

Provenance tracks:
where semantic memory originated.

Provenance supports:

* auditability
* trust
* governance
* explainability
* source weighting

Provenance is:
core semantic infrastructure metadata.

# Core Philosophy

Semantic memory should remain:
traceable to its origins.

The middleware preserves:
source attribution continuously.
