# Phrase Extraction Specification

## Overview

The middleware extracts:
multi-word semantic phrases.

Phrase extraction improves:

* retrieval precision
* semantic specificity
* metadata matching
* business terminology recognition

Phrases are:
first-class retrieval signals.

---

# Core Philosophy

Single keywords often lose:
semantic precision.

Phrase extraction preserves:
contextual meaning.

Examples:

* emergency roof repair
* annual maintenance contract
* kitchen remodeling financing

Phrase-aware retrieval dramatically improves:
high-density semantic retrieval.
