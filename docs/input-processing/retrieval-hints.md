# Retrieval Hint Specification

## Overview

The input processor generates:
structured retrieval hints.

Retrieval hints optimize:
retrieval precision.

Hints support:

* source weighting
* metadata boosting
* relationship expansion
* traversal depth
* compression strategy

Hints improve:
retrieval quality and token efficiency.

---

# Hint Philosophy

The middleware does not allow:
raw user input
to directly drive retrieval behavior.

The input processor generates:
structured retrieval intelligence.
